LMS Quick Guide
  1. Go to Create Module add title, description, and upload learning materials (PDF, DOCX, video, etc.)
  2. Open the module Create Quiz manually or click Generate with AI to auto-generate from uploaded files
  3. Add a Practical Exam with task instructions and evaluation criteria; optionally require webcam
  4. Publish the module so trainees can access it from their portal
  5. Review trainee submissions in Submissions and grade each one (0"“100)
  6. Issue Certificates to trainees who complete all modules
